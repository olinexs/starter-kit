# Ecogreen Design System

## Brand Overview

**Company**: Ecogreen Oleochemicals — oleochemicals manufacturing, Batam.

## Colour Palette

| Token | Value | Usage |
|---|---|---|
| `--eo-primary` | `#4CAF50` | Primary actions, active states |
| `--eo-primary-dark` | `#2E7D32` | Hover, pressed states |
| `--eo-primary-light` | `#81C784` | Backgrounds, chips |
| `--eo-primary-surface` | `#E8F5E9` | Card tints, highlights |
| `--eo-secondary` | `#607D8B` | Secondary actions |
| `--eo-error` | `#F44336` | Errors, delete actions |
| `--eo-warning` | `#FF9800` | Warnings, caution states |
| `--eo-success` | `#4CAF50` | Success states |
| `--eo-text-primary` | `#212121` | Body text |
| `--eo-text-secondary` | `#757575` | Labels, captions |

## Typography

- **Font**: Inter, system-ui, sans-serif
- **Base size**: 14px
- **Heading weight**: 600
- **Body weight**: 400

## Logo Assets

Place in `.design/assets/`:
- `logo.svg` — full logo (for dark/coloured backgrounds)
- `logo-white.svg` — reversed logo (for light backgrounds)
- `favicon.ico` — 32×32 browser icon
